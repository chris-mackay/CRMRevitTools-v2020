Sheet Renamer

Sheet Renamer allows you to rename all PDF files in a directory based on a Sheet Set.

Naming convention is Project Number-Sheet Number_Current Revision

Print: Follow the steps below to setup your print settings

  1. Select your PDF printer from the drop down list in the Printer group.
  2. Select Create separate files in the File group.
  3. Click Browse... and select the directory you want to print to. It's more convenient to create a temporary directory on your Desktop. This directory must contain ONLY the files that you wish to rename.
  4. Select Selected views/sheets in the Print Range group, then click Select... to select your Sheet Set.
  5. Print your sheets as usual

Files before renaming: Your PDF files should look like the ones pictured below



Rename your sheets: Follow the steps below to rename your sheets

  1. Select Browse to open Browse For Folder
  2. Select the directory where the sheets you want to rename are located. Click OK.
  3. Select the Sheet Set you used to print your sheets. Click OK and then Yes to confirm. Your files will be renamed.
